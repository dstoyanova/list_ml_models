# list_ml_models
List ML Models using the STACKn API

## Prerequisites
In models_page/views.py set the domain, username and password. For example:
```python
domain = "34.231.147.144.nip.io"
username = "user"
password = "pass"
```

## Start the app
```bash
python manage.py runserver
```

## Check the result
Go to:
http://localhost:8000/models/
